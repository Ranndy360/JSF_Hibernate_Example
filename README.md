# JSF_Hibernate_Example
JSF web Example CRUD using Hibernate for database and Primefaces for views

# Feature
* CRUD vehicle table

# Resources
* Java 1.8.0
* JDK 1.8
* Netbeans 8.2
* Apache Tomacat 8
* JSF 2.2
* Hibernate 4.3.1
* Mysql 5.0.12 - Xamp


# Install
* Create a BDD name: renta
* Add project to Netbeans
* Add conection to database
* Run as Java Application
